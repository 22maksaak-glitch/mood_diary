🌈 Дневник настроения – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного дневника настроения на 7 языках программирования.
Каждая версия позволяет вести учёт эмоционального состояния, анализировать динамику и сохранять данные.

📝 Что такое Дневник настроения?
Простое приложение для ежедневного отслеживания настроения. Вы записываете своё состояние по шкале от 1 до 10, добавляете заметку и дату/время. Программа помогает:

Анализировать – средняя оценка, минимальное/максимальное значение.

Отслеживать – все записи в хронологическом порядке.

Управлять – добавлять, удалять и просматривать записи.

Сохранять – данные хранятся в файле (JSON) и загружаются при старте.

✨ Особенности
Интерактивное меню – выбор действий по номеру.

Запись содержит: id, дата/время, оценка (1–10), заметка (текст).

Автоинкремент id при создании.

Статистика – средняя оценка, максимум, минимум, количество записей.

Удаление записи по id.

Сохранение / загрузка состояния в файл mood_data.json (или эквивалент).

Цветной вывод (где поддерживается) для наглядности.

Валидация оценки (1–10) и даты.
